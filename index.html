Rql Upload¶

Summary¶

  • Propose a cube to create an upload service from a json file description.
  • Contain a demonstrator that can be generated from scratch.

Description¶

Cube to upload data in a cubicweb CWUpload entity. This cube has no dependency on other cubes. The database schema has been modified to store form parameters in the database and to deport file on the server file system. Some navigation views are designed to simplify the access to the desired forms.
